###   Launch Gazebo   ###
# To launch Gazebo from WSL, start the GUI server.

####   Launch VcXsrv   ####
# Locate and launch the VcXsrv application on the windows machine.
## It shows as "XLaunch" on the Windows Desktop.
# Follow the below settings:
## Set "Display Settings":
### Selection to "Multiple Windows"
### "Display number" to "0"
### Select next
## Set "Client startup":
### Select "Start no client"
### Select next
## Set "Extra Settings":
### Select "Clipboard" and "Primary Selection"
### Unselect "Native opengl"
### Select "Disable access control" 
### Select next
## Select Finish
